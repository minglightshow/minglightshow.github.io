> 强调下，股票市场短期是投票机，赚取的是市场情绪的钱；长期来看却是称重机，赚取的是公司业绩和估值的钱。

> 这里补充下，科技行业是未来的黄金赛道，未来还是有较大的空间。特别是半导体芯片和软件领域。重点关注研发，净利润增速和核心技术这些问题。
